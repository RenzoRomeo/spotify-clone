Spotify clone based on the following technologies:
- Next.js
- React.js
- TailwindCSS
- Recoil
- Next.auth

Spotify's API integration, allows the user to log in and the session persists.

User playlists are automatically imported.

100% responsive design.

Allows the user to:
- Play
- Pause
- Control Volume
